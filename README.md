# README
This is a simple and customizable time traker.

## Requirements

- Git (https://git-scm.com/).
- Docker (https://www.docker.com/get-started).

## Step 1: Clone Repo

```bash
# Clone git repo
git clone git@github.com:brayvasq/jiffy.git

# Move to the repo directory
cd jiffy/
```

## Step 2: Run the project

```bash
docker-compose up
```
